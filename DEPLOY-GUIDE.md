# Deploy Smart Project Hub to Vercel

## Quick Setup (5 minutes)

### 1. Install Vercel CLI
```bash
npm i -g vercel
```

### 2. Login to Vercel
```bash
vercel login
```

### 3. Deploy Your Project
```bash
# From your project directory (c:\wamp64\www\sph)
vercel --prod
```

### 4. Set Environment Variables
```bash
# Generate app key first
php artisan key:generate --force

# Then set in Vercel dashboard or CLI:
vercel env add APP_KEY
vercel env add APP_URL
```

✅ **Free SSL Certificate**

✅ **Global CDN**  
✅ **Git Integration** (push to deploy)

## Custom Domain (Optional)

1. **In Vercel Dashboard**: Go to Project → Settings → Domains
2. **Add your domain**: `yourdomain.com`
3. **Update DNS**: Point to Vercel's nameservers
4. **SSL Certificate**: Auto-generated by Vercel

## Production Checklist

Before deploying, ensure:
- [ ] `.env.example` copied to `.env` with production values
- [ ] `php artisan key:generate --force` run
- [ ] Database configured (SQLite works for Vercel)
- [ ] `npm run build` completed successfully
- [ ] Test all features locally

## Commands

```bash
# Deploy to production
vercel --prod

# Deploy preview (for testing)
vercel

# View logs
vercel logs

# Open project in browser
vercel open
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Build fails | Check `composer install` and `npm run build` locally |
| 500 error | Verify environment variables in Vercel dashboard |
| Database error | Ensure SQLite path is correct in `.env` |
| Assets not loading | Run `npm run build` and redeploy |
